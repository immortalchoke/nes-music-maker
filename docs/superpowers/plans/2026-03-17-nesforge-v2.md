# NESForge v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement 8 UI and interaction improvements to `nesforge.html`: UI scaling, Logic-style piano roll interactions, marquee multi-select, arrangement note previews, loop visibility toggle, insert chord button, popup dismiss fix, and volume bar slider.

**Architecture:** All changes are in `nesforge.html` (single-file app — no build tools, no tests, no bundler). Each task edits a specific region of the file. Tasks are ordered to avoid conflicts: CSS/constants first, then canvas draw routines, then event handlers, then new UI elements.

**Tech Stack:** Vanilla HTML/CSS/JS, Canvas 2D API, Web Audio API, no dependencies except JSZip CDN and Press Start 2P font.

```js
        <button id="ci-toggle" class="nes-btn" style="margin-left:4px;">3</button>
      </div>
      <button id="ci-insert" class="nes-btn" style="width:100%;">INSERT</button>
    `;
    document.body.appendChild(el);

    document.getElementById('ci-toggle').onclick = () => {
      numNotes = numNotes === 3 ? 4 : 3;
      document.getElementById('ci-toggle').textContent = numNotes;
    };

    document.getElementById('ci-insert').onclick = () => {
      const ap = PR.activePattern;
      if (!ap) { alert('Open a pattern in the piano roll first.'); return; }
      const root = document.getElementById('ci-root').value;
      const type = document.getElementById('ci-type').value;
      const rootMIDI = Scales.NOTE_ORDER.indexOf(root) + 60;
      const intervals = CHORD_INTERVALS[type][numNotes];
      const chordNotes = intervals.map(i => rootMIDI + i);
      const entry = ap.track.arrangement.find(en => en.patternId === ap.pattern.id);
      const startBar = entry ? entry.startBar : 0;
      const relBar = ProjectState.playheadBar - startBar;
      const sixteenth = PR.sixteenthWidth() / PR.barWidth();
      const snappedRelBar = Math.round(relBar / sixteenth) * sixteenth;
      const label = root + type;
      ChordTool._insertChord(chordNotes, snappedRelBar, label);
      el.remove();
    };

    setTimeout(() => document.addEventListener('mousedown', () => el.remove(), { once: true }), 100);
  };
```
